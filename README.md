# lab2layer-ae

After Effects 用の音素リップシンクツール

lab ファイル（音素タイミングファイル）を解析して音素レイヤーを生成し、不透明度エクスプレッションを自動設定します

## 概要

lab2layer は、音声合成ソフトが出力する lab ファイルを読み込み、After Effects で口パク（リップシンク）アニメーションを簡単に作成するためのスクリプト
lab ファイルの構造が独自的で特殊だと動かない可能性あり

## 機能

- **lab ファイルの解析**: lab ファイルから音素とタイミング情報を自動抽出
- **音素レイヤーの生成**: マーカー付きのヌルレイヤーを自動作成
- **不透明度エクスプレッション**: 選択したレイヤーに音素連動の不透明度エクスプレッションを自動設定
- **音素フィルタリング**: 必要な音素のみを選択して使用可能
- **複数音素対応**: レイヤー名にカンマ区切りで複数の音素を指定可能

## インストール

1. `lab2layer.jsx` をダウンロード
2. 以下のいずれかの場所に配置:
   - **Windows**: `C:\Program Files\Adobe\Adobe After Effects <version>\Support Files\Scripts\ScriptUI Panels\`
   - **macOS**: `/Applications/Adobe After Effects <version>/Scripts/ScriptUI Panels/`
3. After Effects を再起動
4. メニューから `ウィンドウ` → `lab2layer.jsx` を選択

※ mac 環境での動作チェックはしていないので動かなかったら改変してください

## 使い方

### Step 1: lab ファイルの読み込み

1. スクリプトパネルを開く
2. 「Browse...」ボタンをクリック
3. lab ファイル（`.lab`）を選択
4. 解析された音素がチェックボックス付きで表示される

### Step 2: 音素の選択

| ボタン     | 説明                       |
| ---------- | -------------------------- |
| **All**    | すべての音素を選択         |
| **None**   | すべての選択を解除         |
| **Common** | よく使われる音素を自動選択 |

### Step 3: Phoneme レイヤーの作成

1. 使用する音素にチェックを入れる
2. 「Create Phoneme Layer」ボタンをクリック
3. 「Phoneme」という名前のヌルレイヤーが作成され、各音素がマーカーとして配置される

### Step 4: 口パク画像の準備

各音素に対応する画像レイヤーを用意し、**レイヤー名を音素名に設定**します。

```
例：
- a.png  → レイヤー名: a
- i.png  → レイヤー名: i
- u.png  → レイヤー名: u
- e.png  → レイヤー名: e
- o.png  → レイヤー名: o
- N.png  → レイヤー名: N
```

#### 複数音素を同じ画像で表示する場合

レイヤー名にカンマ区切りで複数の音素を指定できます。

```
例：
- a,e.png → レイヤー名: a,e（「あ」と「え」で同じ口の形を使用）
```

### Step 5: 不透明度エクスプレッションの設定

1. 口パク画像レイヤーをすべて選択
2. 「Setup Opacity」ボタンをクリック
3. 各レイヤーに不透明度エクスプレッションが自動設定される

## 音素例

このスクリプトは特定の音素リストを持っていません（判定していない）

lab ファイルから動的に音素を読み取る仕組みになっています

### 母音

| 音素 | 説明 |
| ---- | ---- |
| a    | あ   |
| i    | い   |
| u    | う   |
| e    | え   |
| o    | お   |

### 特殊な音素

| 音素 | 説明                 |
| ---- | -------------------- |
| N    | 撥音（ん）           |
| cl   | 促音（っ）の閉鎖部分 |
| pau  | ポーズ（休止）       |
| sil  | 無音（silence）      |
| br   | ブレス（息継ぎ）     |

### 子音

| 音素    | 説明             |
| ------- | ---------------- |
| k, g    | か行、が行       |
| s, z    | さ行、ざ行       |
| t, d    | た行、だ行       |
| n       | な行             |
| h, b, p | は行、ば行、ぱ行 |
| m       | ま行             |
| y       | や行             |
| r       | ら行             |
| w       | わ行             |

## lab ファイルってなに？

lab ファイルは以下のように 3 つに区分けされた形式のテキストファイルです

```
0 1000000 sil
1000000 2500000 k
2500000 5000000 o
5000000 7500000 N
7500000 10000000 n
10000000 12500000 i
12500000 15000000 ch
15000000 17500000 i
17500000 20000000 w
20000000 22500000 a
22500000 30000000 sil
```

- 1 列目 → 開始時間（100 ナノ秒単位）
- 2 列目 → 終了時間（100 ナノ秒単位）
- 3 列目 → 音素

## 対応ソフトウェア

上のような音素ファイルならすべて対応しているので、以下ソフトウェアが出力する lab ファイルにも対応しているはず

- VOICEVOX
- NEUTRINO
- CeVIO
- A.I.VOICE
- その他 lab 形式を出力するソフトウェア

## トラブルシューティング

### 「Phoneme layer not found!」エラー

先に「Create Phoneme Layer」ボタンで Phoneme レイヤーを作成してください。

### 口パクが反映されない

レイヤー名が音素名と一致しているか確認してください（大文字小文字も区別されます）。

### lab ファイルが読み込めない

ファイルの文字コードが UTF-8 または Shift-JIS であることを確認してください。
